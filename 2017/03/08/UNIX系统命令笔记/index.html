UNIX 终端命令笔记

2017.03.09 晴

  • pwd 查看当前目录
  • cd 切换目录
    • cd /Users/liutingyuan/hexo/source 绝对目录切换
    • cd hexo 然后cd source层层深入切换
    • cd ../切换至上一级文件夹
    • cd切换至用户名文件夹（eg:liutingyuan）
    • cd /切换至根目录
  • mkdir test新建文件夹
  • touch test.txt新建文件
  • open test.txt打开文件
  • who显示当前用户列表
  • w显示用户及行为信息
  • ls和ls -l显示文件列表，后者信息更详细
  • 修改打开文件的默认应用，eg:设置MacDown为打开.md文件的默认应用：
    • mdls hello-world.md查看文件UTI(文件类型)
    • osascript -e 'id of app "MacDown"'查看软件Bundle ID(应用标识符)
    • duti -s com.uranusjr.macdown net.daringfireball.markdown

  • rm test.txt删除文件

    后续跟新……
